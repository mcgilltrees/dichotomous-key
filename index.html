McGill Campus Tree Finder

This project is built with Next.js and React. To use the full interactive tree identification quiz, clone the repository and run the app locally.

How to run this app locally

  1. Install Node.js (version 18 or newer is recommended).
  2. Clone this repository to your computer.
  3. In a terminal, run npm install.
  4. Then run npm run dev and open the shown URL in your browser.
Hosted via GitHub Pages for the @mcgillTrees project.
